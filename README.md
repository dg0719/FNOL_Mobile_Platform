# ClaimShield FNOL Prototype

This is a Vite + React wrapper around the mobile-friendly FNOL prototype.

## Run locally

```bash
npm install
npm run dev
```

## Deploy with Vercel via GitHub

1. Create a new GitHub repository.
2. Upload all files from this folder.
3. In Vercel, click **Add New → Project**.
4. Import the GitHub repository.
5. Vercel should auto-detect **Vite**.
6. Click **Deploy**.

## Deploy with the Vercel CLI

```bash
npm install
npm i -g vercel
vercel
```

For production deployment:

```bash
vercel --prod
```
